# 🎓 StudentHub

A modern, responsive student productivity platform built as part of a 6-week Full Stack Development Bootcamp.

StudentHub serves as a student's personal academic command center, helping users organize assignments, schedules, goals, and productivity tools in one centralized platform. This project begins as a landing page in Week 1 and will gradually evolve into a complete full-stack web application with authentication, backend APIs, database integration, cloud deployment, Dockerization, and CI/CD pipelines.

## 🎨 Design Decisions

### Why Glassmorphism?

Glassmorphism provides a modern and premium user interface that aligns with current web design trends while maintaining readability and simplicity.

### Why CSS Grid and Flexbox?

CSS Grid is used for structured layouts such as feature cards and statistics sections, while Flexbox provides flexibility for navigation and responsive alignment.

### Why Dark Mode?

Dark mode improves accessibility, reduces eye strain, and enhances the overall user experience.

### Why Animated Counters?

Interactive elements help engage users and create a more dynamic browsing experience.
